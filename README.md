# Voyage Helper

Voyage Helper is a Website to assist you when travelling.

## Installation
Open 2 tabs,

In the 1st one, go in frontend and do:
```
npm install
```
In the 2nd one, go in server and do:
```
npm install
```
Now in the 1st one do :
```
npm start
```
And then in the 2nd one do :
```
node server
```
